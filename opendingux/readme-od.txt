VisualBoy Advance for OD V1.0
--------------------------------------------------------------------------------
This is a GameBoy , GameBoy Color and GameBoy Advance emulator for Open Dingux.
To use this emulator, you must have compatibles ROMS with .gb, .gbc et .gba extension.
Do not ask me about ROMS, I don't have them. A search with Google will certainly help 
you.

How to use vba-od :
--------------------------------------------------------------------------------
Put vba-od.dge to "emulator" folder of inner flash-memory of your handheld.
Put your .gb, .gbc, .gba games where you want.

Controls :
 * Direction pad : GameBoy & GameBoy Advance pad
 * A is button A
 * B is button B
 * START is button START
 * SELECT is button SELECT
 * X is button A
 * Y is button B
 * R is button R
 * L is button L
 
Use START + SELECT to enter menu. 
During menu browsing, RIGHT and LEFT can go through pages of available games.
